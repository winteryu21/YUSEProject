# 로그라이크 프로젝트 계획

## 1. 프로젝트 개요
* **기간:** 2025년 11월 13일 ~ 2025년 12월 5일 (총 23일, 주말 포함)
* **팀원:** 6명
* **목표:** SDS 기반의 핵심 기능이 완성된 빌드 제작
* **핵심 전략:** 4개의 짧은 스프린트로 나누어 매주 통합 및 테스트를 진행. 각 팀원은 5대 기능 패키지에 배분되어 병렬로 작업을 수행함.

## 2. 5대 기능 패키지 (Work Packages)
* **[패키지 1] 코어 백엔드:** `GameManager`, `Input`, `Audio`, `Data Save/Load` (A-2, A-3, A-4, E-3)
* **[패키지 2] 플레이어 로직:** 플레이어 동작, 성장, 장비 로직 (B-1, B-2, B-3)
* **[패키지 3] 적/월드 로직:** 몬스터 스폰, AI, 퀘스트 로직 (C-1, C-2, C-3)
* **[패키지 4] 인게임 UI/UX:** HUD, 패널, 보상 시스템 (D-1, D-2, B-4)
* **[패키지 5] 메타 UI/UX:** 메인 메뉴, 설정, 강화, 도감 (E-1, E-2, A-5)

> A-2와 같은 번호는 작업 분류를 의미, 제일 하단의 4번 목차 참고!
